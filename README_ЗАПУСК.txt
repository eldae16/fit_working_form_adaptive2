Как запустить проект:

1. Установить зависимости:
   pip install -r requirements.txt

2. Применить миграции базы данных:
   python manage.py migrate

3. Запустить сервер:
   python manage.py runserver

Страницы проекта:
- / или /index.html — главная страница с рабочей формой записи
- /services/ или /services.html — каталог услуг
- /contacts/ или /contacts.html — контакты с картой и форма записи
- /reviews/ или /reviews.html — отзывы клиентов
- /cabinet/ или /cabinet.html — личный кабинет

Для входа/регистрации используется номер телефона как логин.

Форма записи на главной и странице контактов сохраняет заявки в базу данных. Посмотреть заявки можно в админ-панели Django.
